Comparación de algoritmos SSSP — UNSAAC, Algoritmos Avanzados, Grupo 2


## Segundo paper: BMSSP en Lightning Network

Para el segundo paper se agregaron dos repositorios base:

- `implementacion-2-rap2363-ssps`: contiene la implementación en Rust de BMSSP y Dijkstra usada para ejecutar pruebas sobre grafos reales.
- `implementacion-2-ln-bmssp-evaluation`: contiene el notebook de análisis experimental asociado al paper de BMSSP aplicado a la Lightning Network.

El objetivo de esta parte es comparar el rendimiento práctico de BMSSP frente a Dijkstra, usando los resultados reportados por los autores y los resultados obtenidos en una laptop propia.
